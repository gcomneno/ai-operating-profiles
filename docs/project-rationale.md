# Project rationale

AI assistants are often customized with vague preferences such as "be concise" or "be helpful".

This lab explores a more operational approach: custom instructions as reusable workflow profiles.

The goal is to collect practical instruction patterns that define how an assistant should collaborate with a human in real tasks, especially when precision, learning, and step-by-step execution matter.

## Starting point

The first profile, Scimmietta Operativa, focuses on terminal-first technical workflows.

It is designed for users who want to:

- execute one command at a time;
- understand what each step does;
- avoid unnecessary formatting friction;
- prevent assistants from guessing unseen command outputs;
- generate useful handoff recaps when a conversation becomes too long.

## Principle

A good AI profile should not merely make an assistant sound pleasant.

It should make the assistant operationally compatible with the person using it.
